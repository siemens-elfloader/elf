Console v 0.3


Запуск эльфов/шорткатов/энтриков из текстовой строки с передачей параметров.
Алиасы добавляются командой add непосредственно из консоли.

add mc 0:\zbin\mc\mc.elf
add call 0:\zbin\utilities\console\call.elf
add alarm a0bb359f
add mainmenu CKLK_CK_MAINMNU
add note "Выпить йаду"

И запускаются как 

mс
call *100#
alarm
mainmenu
note

Исполнение команды - зеленая кнопка. 

Также имеются "вшитые" команды: add, halt, clear, quit, exit, list.
add - добавление алиасов (см. выше)
clear - очистка лога
halt/quit/exit - выход
list - список алиасов

Имеется возможность писать "консольные приложения". Смотрим сорцы утилит call и disk в архиве

Cmd v0.1

Запуск эльфов/шорткатов/энтриков из текстовой строки с передачей параметров.
Алиасы прописываются в файле 0:\zbin\etc\run.txt в формате:

ALIAS: mc 0:\zbin\mc\mc.elf
ALIAS: call 0:\zbin\utilities\console\call.elf
ALIAS: alarm a0bb359f
ALIAS: mainmenu CKLK_CK_MAINMNU
....

Или добавляются командой add непосредственно из консоли.
Пример: add mc 0:\Zbin\mc\mc.elf

"Вшитые" команды: add, halt, clear, quit, exit, bootcoreformat.

В архиве также имеется демонстрационная утилита call - звонки из консоли
Пример: 

add call 0:\Zbin\utilities\console\call.elf
call +79111234567 
- звонок по номеру +79111234567

Ах да..софты не работают. Исполнение команды - зеленая кнопка.

Известные баги:
 - лог очень легко засрать в хлам
 - нифига не работает
 - совсем нифига не работает
 - ну ваааще нифига не работает
 - при запуске эльфа надо нажать джой вниз (фокус ставится не на первый элемент диалога, а на второй)
