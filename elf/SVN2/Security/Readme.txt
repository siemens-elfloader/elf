Проект противоугонного патча для S75.
Данная конфигурация (на момент выкладывания на SVN)
расчитана на 47-ю версию прошивки.
• SecPatch   - собственно патч, в папке Release\Exe находятся скрипты, выдающие *.vkp
• SecCaller  - простенький эльф, вызывающий тело патча в целях отладки.
• IMSIReader - эльф для чтения текущего IMSI, сохраняет IMSI в файле 0:\Misc\IMSI.txt

В папку !Utils надо положить ФФ для создания *.vkp
